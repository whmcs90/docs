---
title: 'Live QR Status'
openapi: 'GET /instance/qr/live'
---

Open a live QR page that automatically changes to a success message after the WhatsApp instance is connected.

This endpoint returns an HTML page with a QR image and a small status poller. Use it when you want the QR display to update automatically after the user scans the code.

```html
<iframe
  src="https://app.wachat.net/api/v1/instance/qr/live?instanceName=t1_mq9r2jov_store-jakarta&apiKey=YOUR_API_KEY"
  width="420"
  height="460"
></iframe>
```

<Warning>
  Passing an API key in a URL can expose it in browser history, logs, or analytics. Use this only for trusted internal dashboards or documentation previews.
</Warning>
